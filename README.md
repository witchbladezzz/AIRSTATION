# AIRSTATION
IoT-станция мониторинга качества воздуха — компактное устройство на базе ESP32-C3, которое измеряет параметры воздуха в реальном времени и передаёт их в облако.

Что измеряет:

Температура и влажность (AHT21)TVOC и eCO₂ (ENS160)Мелкодисперсная пыль PM1.0 / PM2.5 / PM10 (PMS7003)GPS-координаты (NEO-7M)
Данные:

Отправляются в облачную платформу Rightech (gsm либо при  отсутствии gsm wifi)Доступны в реальном времениМожно строить тепловую карту загрязнения города
Применение - установка системы на общественный транспорт, мониторинг вышеуказанных параметров, и их отправка в облачную платформу.

Питание:

ESP32-C3 через USB - С.
Устройство показывает качество воздуха в любой точке и помогает выявлять зоны с повышенным загрязнением.
